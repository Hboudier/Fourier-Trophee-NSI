Notre Projet


Ce dessin est un dessin qui utilise les series de Fourrier pour dessiner un train.


Il utilise des epicycles pour dessiner les differentes lignes du train.


Il est possible d'ajuster le nombre d'epicycles avec la barre sous le dessin.
