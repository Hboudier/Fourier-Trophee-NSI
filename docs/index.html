SpiroDraw


Ce projet utilise les series de Fourrier pour dessiner des silhouettes.


Il est possible d'ajuster le nombre d'epicycles avec la barre sous le dessin ainsi que de changer la forme dessinée.


Pour plus d'explications sur le fonctionnement du programme, regarder la page "Explication"


Pour pouvoir faire son propre dessin: choisir la forme "user input" et maintenir la touche "shift" appuyé lors du dessin
